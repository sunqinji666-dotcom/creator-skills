# Creator Skills

Reusable, privacy-aware skills and workflows for creative production.

![creator-skills project visual](assets/creator-skills-hero.png)

## What it does

Reusable, privacy-aware skills and workflows for creative production. It is designed to be understandable, local-first where applicable, and configurable without committing credentials.

## Quick start

Read the project README for the platform-specific command. Keep secrets in environment variables or local configuration files that are ignored by Git.

## Privacy boundary

The public repository contains no personal captures, credentials, tokens, or deployment secrets. Supply your own data locally.
